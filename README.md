# Noore-Soha Order Form

راهنمای سریع:
1. `npm install`
2. `.env.local` با متغیرهای `TELEGRAM_BOT_TOKEN`, `TELEGRAM_CHAT_ID`, `NEXT_PUBLIC_WHATSAPP_NUMBER` بسازید.
3. `npm run dev` سپس باز کنید `http://localhost:3000`
4. برای دیپلوی: پوش کردن به GitHub و Import در Vercel، تنظیم Environment Variables در Vercel.
